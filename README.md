## :warning: Instalação <a name="install"></a>

```bash
# Instalar todas as dependências necessárias:
$ npm install

# É preciso criar e configurar o SGBD em:
$ /src/config/db.js

# Iniciar o projeto em localhost:
$ npm start

```
